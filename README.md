# My personal homepage

Powered by [Jekyll](http://jekyllrb.com/), [Github Pages](https://pages.github.com/) and others.


## USAGE

Install dependencies with bundle:
```bash
bundle install
```

Then run locally:

```bash
bundle exec jekyll serve
```
